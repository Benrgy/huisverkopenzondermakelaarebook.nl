## 📋 Project Overzicht

Deze repository bevat een comprehensive website die Nederlandse huiseigenaren helpt bij het **huis verkopen zonder makelaar**. De gids biedt praktische stappen, juridische informatie en kostenanalyses specifiek voor de Nederlandse woningmarkt van 2024-2025.

### 🎯 Hoofddoelstellingen

- **Kostenbesparing**: Gemiddeld €2.250-€3.945 besparen op makelaarcommissies
- **Praktische begeleiding**: Stap-voor-stap proces voor particuliere verkoop
- **Juridische compliance**: Nederlandse wet- en regelgeving voor woningverkoop
- **Marktinzicht**: Actuele data van de Nederlandse vastgoedmarkt

## 🏡 Voor Wie Is Deze Gids?

### Nederlandse Huiseigenaren
- Eigenaren die **zelf hun woning verkopen** willen in Nederland
- Mensen die **makelaarkosten willen besparen** (gemiddeld 1,12-1,16% van verkoopprijs)
- Particuliere verkopers die juridische zekerheid zoeken
- Tech-savvy eigenaren die digitale platforms willen gebruiken

### Belgische Huiseigenaren
- Nederlandstalige eigenaren in België die particuliere verkoop overwegen
- Grensstreek bewoners die Nederlandse marktinformatie willen

### Kostenvergelijking Traditioneel vs Particulier

| Verkoopmethode | Kosten bij €450.000 | Besparing |

## 🚀 Lokale Ontwikkeling

### Vereisten
- Ruby 2.7+
- Jekyll 4.0+
- Git

### Installatie

```bash
# Clone repository
git clone https://github.com/benrgy/huisverkopenzondermakelaarebook.nl.git

# Navigeer naar directory
cd huisverkopenzondermakelaarebook.nl

# Installeer dependencies
bundle install

# Start lokale server
bundle exec jekyll serve

# Open browser
open http://localhost:4000
```

### Development Commands

```bash
# Build site
bundle exec jekyll build

# Serve with live reload
bundle exec jekyll serve --livereload

# Build for production
JEKYLL_ENV=production bundle exec jekyll build
```
